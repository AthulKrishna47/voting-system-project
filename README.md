# Election Voting System (12th Grade Project)

A Python-based console application that simulates an election voting system connected with a MySQL database. Users can authenticate with an admission number, vote for SPL and House Captain candidates, and view election results.

## Features

- User login authentication by admission number
- Voting for SPL candidates
- Voting for House Captain candidates by house
- Prevents double voting
- Displays election winners for SPL and House Captain positions
- Uses MySQL for persistent vote storage

## Technologies Used

- Python 3
- MySQL Database
- mysql-connector-python library

## Setup Instructions

1. Ensure you have Python 3 and MySQL installed on your machine.
2. Create a MySQL database and required tables: `voters`, `candidate`, and `votes`.
3. Populate tables with data relevant to your election.
4. Replace the database connection placeholders in the code: username, password, database name.

6. Follow on-screen prompts to login and vote.
